Portfólio do Darlan Kennedy
===========================

Como publicar (GitHub Pages):
1) Faça upload dos arquivos `index.html`, `profile.jpg` e `cv.pdf` para o repositório.
2) Em Settings → Pages, escolha a branch (ex.: `main`) e a pasta `/root`.
3) Aguarde a publicação (URL do GitHub Pages).

Personalizações rápidas:
- Troque `profile.jpg` (sua foto) e `cv.pdf` (seu currículo).
- Para destacar projetos específicos do GitHub, edite no <script> do HTML:
  const FEATURED_REPOS = ['owner/repo1','owner/repo2']
  // ou use topic:
  const FEATURED_TOPIC = 'portfolio'
